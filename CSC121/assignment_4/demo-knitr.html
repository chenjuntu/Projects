This is a demonstration script that is used for showing how the knitr::spin function can be used to run an R script file, producing an HTML file that contains

  • The R commands from the script
  • The output of these commands
  • All the plots produced by these commands

This HTML file can then be viewed with a browser. This procedure has two advantages:

  1. All the results get put in a single file.
  2. You can put in “markdown” comments, like these, that become nicely formatted text, with bullet points like above, or numbered points like here.

> s <- sd(data)

> # Plot the data points, along with a horizontal line at the mean, two

>
